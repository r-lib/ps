# Suspend (stop) the process

Suspend process execution with `SIGSTOP` preemptively checking whether
PID has been reused. On Windows this has the effect of suspending all
process threads.

## Usage

``` r
ps_suspend(p = ps_handle())
```

## Arguments

- p:

  Process handle or a list of process handles.

## Examples

``` r
px <- processx::process$new("sleep", "10")
p <- ps_handle(px$get_pid())
p
#> <ps::ps_handle> PID=9186, NAME=sleep, AT=2026-03-31 11:52:18.05
ps_suspend(p)
#> NULL
ps_status(p)
#> [1] "stopped"
ps_resume(p)
#> NULL
ps_status(p)
#> [1] "sleeping"
ps_kill(p)
#> [1] "terminated"
```
